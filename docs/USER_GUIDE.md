# Elysium Agent User Guide

## Getting Started

### Creating Your Account

1. Visit the Elysium Agent platform
2. Click **"Sign In with Manus"**
3. Complete Manus OAuth authentication
4. Your account is automatically created

### First Login

Upon first login, you'll see the **Dashboard** with:

- **Recent Tasks**: Your latest task executions
- **Statistics**: Active tasks, completed tasks, total conversations
- **Quick Actions**: Buttons to create new tasks or access documents

---

## Creating and Managing Tasks

### Creating a New Task

1. Click **"Create New Task"** from the dashboard or sidebar
2. Enter a **Task Title** (e.g., "Research AI Trends")
3. Provide a detailed **Objective** describing what you want the agent to accomplish
4. Click **"Create Task"**

The agent will analyze your objective and create an execution plan.

### Task Lifecycle

| Status | Description |
|--------|-------------|
| **Pending** | Task created, waiting for execution |
| **Running** | Agent is actively working on the task |
| **Completed** | Task finished successfully |
| **Failed** | Task encountered an error |
| **Cancelled** | Task was manually stopped |

### Viewing Task Details

1. Click on a task from the dashboard or recent tasks list
2. View the **Task Workspace** with:
   - **Chat Interface**: Conversation history with the agent
   - **Progress Tracker**: Step-by-step execution status
   - **Task Information**: Title, objective, status, and results
   - **Execution Controls**: Plan, Execute, or Cancel buttons

### Monitoring Progress

The progress tracker shows:

- **Current Step**: What the agent is working on
- **Step Status**: Pending, running, completed, or failed
- **Progress Bar**: Overall task completion percentage
- **Error Messages**: Details if a step fails

---

## Chat Interface

### Sending Messages

1. Type your message in the chat input box
2. Press **Enter** or click **Send**
3. The agent will respond with analysis or next steps

### Message Types

- **User Messages**: Your questions or instructions
- **Agent Messages**: AI responses with analysis and results
- **System Messages**: Status updates and notifications

### Markdown Rendering

Agent responses support full **Markdown formatting**:

- **Bold** and *italic* text
- Lists and code blocks
- Links and images
- Tables and quotes

---

## Document Management

### Accessing Documents

1. Click **"Documents"** in the sidebar
2. View all your documents organized by:
   - **Type**: Generated by agent or uploaded by you
   - **Date**: Most recent first
   - **Task**: Associated task (if any)

### Uploading Documents

1. In the **Document Center**, click **"Upload Document"**
2. Select files from your computer
3. Documents are automatically stored and indexed

### Downloading Documents

1. Find the document in the list
2. Click the **Download** icon
3. File is saved to your downloads folder

### Document Search

Use the search bar to find documents by:

- **Filename**: Exact or partial match
- **Type**: Filter by document type
- **Date Range**: Filter by upload/generation date

## Settings and Preferences

### Profile Management

1. Click **"Settings"** in the sidebar
2. Go to the **Profile** tab
3. View your account information:
   - Name and email
   - Account role (User or Admin)
   - Member since date

### Notification Preferences

1. Go to **Settings** → **Notifications**
2. Toggle preferences for:
   - **Task Updates**: Notifications when tasks complete or fail
   - **Email Notifications**: Receive email summaries

### API & Usage

1. Go to **Settings** → **API & Usage**
2. View your **API Key** for programmatic access
3. Monitor your **Usage Statistics**:
   - API calls used this month
   - Tasks executed
   - Storage used

## Admin Panel (Admin Users Only)

### Accessing Admin Panel

1. Click **"Admin"** in the sidebar (visible only to admins)
2. View the admin dashboard

### User Management

1. Go to **Admin** → **Users**
2. View all registered users with:
   - Name and email
   - Account role
   - Registration date
   - Last login

### Managing User Roles

1. Find the user in the list
2. Click the **role dropdown**
3. Select **"User"** or **"Admin"**
4. Changes take effect immediately

### System Statistics

View platform-wide statistics:

- **Total Users**: Registered user count
- **Active Tasks**: Currently running tasks
- **Completed Tasks**: Total successful executions
- **Failed Tasks**: Tasks with errors

## Tips and Best Practices

### Writing Effective Objectives

**Good Objective:**
> "Research the top 5 AI trends in 2024, summarize each trend in 2-3 sentences, and create a formatted document with findings."

**Poor Objective:**
> "Tell me about AI"

### Breaking Down Complex Tasks

For complex objectives, provide context:

- **Background**: What problem are you solving?
- **Requirements**: Specific outputs or formats needed
- **Constraints**: Time limits, budget, or other restrictions
- **Success Criteria**: How will you know the task succeeded?

### Monitoring Long-Running Tasks

For tasks taking more than a few minutes:

1. Keep the browser tab open
2. Check progress regularly
3. Monitor the chat for updates
4. Review step-by-step execution

### Handling Failed Tasks

If a task fails:

1. Review the error message
2. Check the chat history for context
3. Modify the objective if needed
4. Create a new task with improvements

## Troubleshooting

### Task Stuck in "Running" Status

1. Wait 5-10 minutes for completion
2. If still running, click **"Cancel Task"**
3. Review the error message
4. Try again with a simpler objective

### Missing Documents

1. Check the **Document Center** search
2. Filter by date range
3. Verify the document was generated by the task
4. Contact support if still missing

### Chat Not Responding

1. Refresh the page (Ctrl+R or Cmd+R)
2. Check your internet connection
3. Try sending a simple message
4. Contact support if issue persists

### Permission Denied Errors

1. Verify you're logged in
2. Check if the task belongs to you
3. Contact an admin if you need access
4. Admin users can access all tasks

## Frequently Asked Questions

**Q: How long do tasks typically take?**
A: Most tasks complete within 5-30 minutes depending on complexity.

**Q: Can I run multiple tasks simultaneously?**
A: Yes, you can create and run multiple tasks at the same time.

**Q: Are my documents stored permanently?**
A: Yes, documents are stored indefinitely unless you delete them.

**Q: Can I export my data?**
A: Yes, you can download documents and export conversation history.

**Q: What happens if a task fails?**
A: You'll receive an error notification. Review the error and try again with adjustments.
